Algunos de los proyectos que e realizado mientras he estado cursando el grado de Desarrollo de Aplicaciones Multiplataformas son los siguientes:

Aviator Candán | HTML-CSS-JavaScript

Este proyecto consistio en realizar un pequeño juego usando HTML, CSS y JavaScript. El desarrollo se centró en la implementación de lógica de colisiones, gestión de estados del juego y renderizado en tiempo real utilizando únicamente tecnologías web nativas, sin librerías externas. Este enfoque permite un rendimiento fluido y una experiencia accesible desde cualquier navegador moderno.

Puente Fesac | C++

Este proyecto fue desarrollado en C++ como una aplicación de consola, centrada en la práctica de estructuras de control, manejo de variables y lógica de programación.
